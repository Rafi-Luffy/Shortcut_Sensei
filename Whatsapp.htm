Top 93 Shortcuts for Whatsapp

Welcome to WhatsApp Shortcuts!

WhatsApp is one of the most popular messaging applications worldwide, offering users a seamless experience for text messaging, voice, and video calls. But did you know that you can make your WhatsApp experience even more efficient with keyboard shortcuts? Whether you are a power user or someone who wants to save a few seconds with every interaction, these shortcuts can greatly enhance your productivity.


Why Use Shortcuts?

  • Speed : Perform tasks faster without navigating through menus.
  • Efficiency : Streamline your workflow and reduce repetitive actions.
  • Productivity : Focus more on your conversations and less on finding features.

Explore the comprehensive list of WhatsApp shortcuts below, sorted into categories for easy reference. Master these shortcuts to become a WhatsApp pro!

Chatting Shortcuts

These shortcuts help you navigate through your chats efficiently, allowing you to quickly switch between conversations, search for specific messages, and manage your chat history.

Key Combination Description
Ctrl + N New Chat
Ctrl + F Search
Ctrl + P Previous Chat
Ctrl + N Next Chat
Ctrl + Shift + M Mute Chat
Ctrl + Shift + H Archive Chat
Ctrl + Shift + D Delete Chat
Ctrl + Shift + U Mark as Unread
Ctrl + R Reply to Message
Ctrl + C Copy Selected Message
Ctrl + X Cut Selected Message
Ctrl + V Paste Message

Formatting Shortcuts

With these shortcuts, you can format your messages easily by making text bold, italic, or strikethrough, changing font size and color, and more, enhancing the visual appeal of your messages.

Key Combination Description
Ctrl + B Bold Text
Ctrl + I Italic Text
Ctrl + S Strikethrough Text
Ctrl + Shift + M Monospace Text
Ctrl + Shift + + Increase Font Size
Ctrl + Shift + - Decrease Font Size
Ctrl + Shift + C Change Font Color
Ctrl + Shift + B Change Background Color

Navigation Shortcuts

These shortcuts streamline your navigation within the WhatsApp interface, enabling you to jump to the beginning or end of your chat list, scroll through chats, and navigate through your chat tabs swiftly.

Key Combination Description

Home Scroll to Top
