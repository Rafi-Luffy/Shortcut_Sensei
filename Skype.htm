Top 75 Shortcuts for Skype

Welcome to Shortcut Sensei!

Ready to supercharge your Skype experience? Whether you're making calls, hosting meetings, or chatting with friends, mastering Skype shortcuts can save you valuable time and make your interactions smoother.


Why Use Shortcuts?

  • Speed : Perform tasks faster without navigating through menus.
  • Efficiency : Streamline your workflow and reduce repetitive actions.
  • Productivity : Focus more on your conversations and less on finding features.

Explore the comprehensive list of Skype shortcuts below, sorted into categories for easy reference. Master these shortcuts to become a Skype pro!

General Shortcuts

Master the basics of Skype with these essential shortcuts. From starting calls to managing contacts, these shortcuts cover the fundamental functions you need for smooth communication.

Shortcuts Description
Ctrl + Comma (Ctrl + ,) Open Settings
Esc Close active window/popup
Ctrl + Q Quit Skype
Ctrl + Shift + Q Sign out
Ctrl + H Show Skype Home
Ctrl + 1 Open Contacts
Ctrl + 2 Open Calls
Ctrl + 3 Open Chats
Ctrl + 4 Open Notifications
Ctrl + N Start a new chat
Ctrl + Tab Switch between chats
Ctrl + Shift + Tab Switch back between chats
Ctrl + Enter Send message
Shift + Enter Insert a new line in a chat
Ctrl + F Find in current chat
Ctrl + Shift + F Search Skype
Alt + 1 Switch to Contacts view
Alt + 2 Switch to Calls view
Alt + 3 Switch to Chats view
Alt + 4 Switch to Notifications view

Messaging and Formatting Shortcuts

Enhance your messages with these Skype shortcuts for formatting. Easily apply bold, italic and other text styles to your messages for clearer communication. Quickly mark messages as read or unread to stay organized and on top of your conversations.

Shortcuts Description
Ctrl + E Focus on text input in the chat box
Ctrl + L Mark all messages as read
Ctrl + Shift + U Mark selected messages as unread
Ctrl + B Apply bold formatting
Ctrl + I Apply italic formatting
Ctrl + Shift + X Apply strikethrough formatting
Ctrl + Shift + 7 Apply list formatting
Ctrl + Shift + M Apply monospaced font
