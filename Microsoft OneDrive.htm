Navigation Shortcuts

These shortcuts help you navigate through OneDrive quickly, allowing you to open new windows, search for files, and move between locations effortlessly.

Key Combination Description
Ctrl + N Open a new OneDrive window.
Ctrl + W Close the current OneDrive window.
Ctrl + F Search for files or folders within OneDrive.
Ctrl + O Open selected file or folder.
Alt + Left Arrow Go back to the previous location.
Alt + Right Arrow Go forward to the next location.
Ctrl + Shift + E Open OneDrive Explorer view.
Ctrl + Shift + P Toggle preview pane.
Ctrl + Shift + T Open a new tab in the current OneDrive window.
Ctrl + Tab Switch between open tabs in OneDrive.

Selection Shortcuts

Use these shortcuts to efficiently select multiple files or folders within OneDrive, whether they are contiguous or scattered across the window.

Key Combination Description
Ctrl + Shift + A Select all files and folders within the current folder.
Ctrl + Click Select multiple non-contiguous items.
Shift + Click Select a range of items.
Ctrl + Space Select the currently highlighted item.

File Management Shortcuts

Streamline your file management tasks with these shortcuts, including copying, cutting, pasting, deleting, undoing actions, redoing actions, and duplicating items.

Key Combination Description
Ctrl + C Copy selected item(s).
Ctrl + X Cut selected item(s).
Ctrl + V Paste copied or cut item(s).
Delete Delete selected item(s).
Shift + Delete Delete item(s) permanently without moving to the Recycle Bin.
Ctrl + Z Undo the last action.
Ctrl + Y Redo the last undone action.
Ctrl + D Duplicate selected item(s).

File/Folder Navigation Shortcuts

Navigate through your files and folders with ease using these shortcuts, which include jumping to the top or bottom of the page, moving up or down in the folder hierarchy, renaming items, and viewing item properties.

Key Combination Description
Home Jump to the top of the current page.
End Jump to the bottom of the current page.
Up Arrow Move up one folder level.
Backspace Go up one folder level.
F2 Rename selected item.
Alt + Enter View properties of the selected item.
Ctrl + Shift + C Collapse all folders in the current view.

Sharing and Collaboration Shortcuts

Collaborate effectively with others by using these shortcuts to share files, email items, save copies, follow items for updates, remove sharing links, move items, and view previous versions of files.

Key Combination Description
Ctrl + S Share the selected item(s).
Ctrl + E Email the selected item(s).
Ctrl + Shift + S Save a copy of the selected item(s).
Ctrl + Shift + F Follow the selected item(s) (receive notifications for changes).
Ctrl + Shift + R Remove the sharing link.
Ctrl + Shift + M Move the selected item(s) to another location.
Ctrl + Shift + H Show previous versions of the selected file.

Sync Shortcuts

Manage OneDrive syncing efficiently with these shortcuts, which allow you to pause and resume syncing as needed.

Key Combination Description
Ctrl + Shift + D Pause syncing.

View Options Shortcuts

Customize your OneDrive view with these shortcuts, including options to switch between different icon sizes, view types, and hide or show navigation and details panes.

Key Combination Description
Ctrl + 1 Switch to extra-large icons view.
Ctrl + 2 Switch to large icons view.
Ctrl + 3 Switch to medium icons view.
Ctrl + 4 Switch to small icons view.
Ctrl + 5 Switch to list view.
Ctrl + 6 Switch to details view.
Ctrl + 7 Switch to tiles view.
Ctrl + 8 Switch to content view.
Ctrl + Shift + V Show or hide the navigation pane.
Ctrl + Shift + T Show or hide the details pane.
Ctrl + Shift + R Refresh the current view.

Search Shortcuts

Quickly find what you're looking for in OneDrive using these search shortcuts, which enable you to activate the search box, open the search pane, search within folders, and view search history.

Key Combination Description
Ctrl + E Activate the search box.
F3 Open the search pane.
Ctrl + F Search within the current folder.
Ctrl + Shift + F Search within the entire OneDrive.
Alt + D Go to the search box.
Ctrl + Shift + Q Show search history.

Miscellaneous Shortcuts

These shortcuts cover various other functions in OneDrive, such as printing items, closing windows, refreshing views, navigating up in the folder hierarchy, opening the Start menu, switching between open windows, accessing menus, opening Task Manager, and signing out of OneDrive.

Key Combination Description
Ctrl + P Print the selected item(s).
Alt + F4 Close the current window.
F5 Refresh the current view.
Alt + Up Arrow Go up one level in the folder hierarchy.
Ctrl + Esc Open the Start menu.
Alt + Tab Switch between open windows.
Alt + F Open the File menu.
Alt + E Open the Edit menu.
Alt + V Open the View menu.
Ctrl + Shift + Esc Open Task Manager.
Ctrl + Shift + Q Sign out of OneDrive.
