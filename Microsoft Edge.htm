Shortcuts for Microsoft Edge

Welcome to Shortcut Sensei! Here, you'll discover a collection of essential keyboard shortcuts tailored for optimizing your browsing experience in Microsoft Edge. Whether you're a seasoned user or just starting, these shortcuts will help you navigate with speed and efficiency, making your web browsing more productive than ever before.


Microsoft Edge, the sleek and powerful web browser from Microsoft, offers a multitude of features to enhance your online journey. Explore our curated list of shortcuts to unlock its full potential, from seamless navigation and efficient tab management to enhanced productivity tools for developers. Start mastering these shortcuts today and elevate your browsing experience to new heights.


Navigation

Mastering navigation shortcuts in Microsoft Edge can greatly enhance your browsing efficiency. With simple key combinations, you can quickly move between tabs, access the address bar, and open new windows. These shortcuts streamline your workflow, making it easy to switch contexts and manage multiple tasks seamlessly.

Key Combination Description
Ctrl + L Selects the URL in the address bar for editing
Ctrl + N Opens a new window
F9 Activates Immersive view
Ctrl + O Opens a file from your computer in Edge
Ctrl + P Prints the current page
Ctrl + S Saves the current page
Ctrl + Shift + L Pastes and searches or pastes and goes (if it's a URL)
Ctrl + Shift + M Allows signing in as a different user or browsing as a Guest
Ctrl + Shift + P Prints using the system dialog
Ctrl + Shift + V Pastes without formatting
Ctrl + Shift + W Closes the current window
Ctrl + Enter Adds www. to the beginning and .com to the end of text typed in the address bar
Ctrl + Shift + Del Opens clear browsing data options
Alt Sets focus on the Settings and more "…" button
Alt + D Selects the URL in the address bar for editing
Alt + E or Alt + F Opens the Settings and more "..." menu
Alt + Shift + I Opens the Send feedback dialog
Alt + Shift + T Sets focus on the first item in the toolbar
Alt + Left arrow Goes back
Alt + Right arrow Goes forward
F1 Opens Help
F4 Selects the URL in the address bar
Ctrl + F4 Closes the current tab
F6 Switches focus to next pane
Shift + F6 Switches focus to previous pane
F7 Turns caret browsing on or off
F10 Sets focus on the Settings and more "…" button
F10 + Enter Opens Setting and more "…" menu
Shift + F10 Opens context menu
F11 Enters full screen (toggle)
Esc Stops loading page; closes dialog or pop-up
Space Scrolls down webpage, one screen at a time
Shift + Space Scrolls up webpage, one screen at a time
Page Down Scrolls down webpage, one screen at a time
Page Up Scrolls up webpage, one screen at a time
Home Goes to the top of the page; moves keyboard focus to first item of pane
End Goes to the bottom of the page; moves keyboard focus to last item of pane
Ctrl + Shift + U Reads aloud (text to speech)

Search and Find

Efficiently locate words or phrases on webpages with Microsoft Edge shortcuts, enhancing browsing productivity. Master these commands for seamless navigation.

Shortcut Description
Ctrl + E Opens a search query in the address bar
Ctrl + F or F3 Finds text on the page
Ctrl + G Jumps to the next match to your Find Bar search
Ctrl + Shift + G Jumps to the previous match to your Find Bar search
Ctrl + Shift + L Pastes and searches or pastes and goes (if it's a URL)

Tabs

Effortlessly manage multiple tabs in Microsoft Edge with intuitive shortcuts, streamlining your browsing experience. Quickly switch between tabs and organize your workflow for optimal productivity.

Shortcut Description
Tab Go to next tab stop
Shift + Tab Go to previous tab stop
Ctrl + H Open History in a new tab (web UI)
Ctrl + J Open Downloads in a new tab (web UI)
Ctrl + M Mute current tab (toggle)
Ctrl + Shift + N Open a new InPrivate window
Ctrl + T Open a new tab and switch to it
Ctrl + W Close the current tab
Ctrl + 1 … to 8 Switch to a specific tab
Ctrl + 9 Switch to the last tab
Ctrl + Tab or Ctrl + Page Down Switch to the next tab
Ctrl + Shift + Tab or Ctrl + Page Up Switch to the previous tab
Alt + Home Open your home page in the current tab
Ctrl + F4 Close the current tab

Zooming

### Zooming Adjust page magnification swiftly in Microsoft Edge with keyboard shortcuts, optimizing readability and viewing convenience.

Shortcut Description
Ctrl + 0 Reset zoom level
Ctrl + + Zoom in
Ctrl + - Zoom out

Reloading

Refresh webpages instantly in Microsoft Edge with convenient shortcuts, ensuring up-to-date content without interrupting your browsing flow.

Shortcut Description
Ctrl + Shift + R or Shift + F5 Reload the current page, ignoring cached content
Ctrl + Shift + T Reopen the last closed tab, and switch to it
F5 or Ctrl + R Reload the current page

In a PDF

Effortlessly navigate PDFs in Microsoft Edge with intuitive shortcuts, enabling seamless scrolling, zooming, and searching for efficient document handling.

Shortcut Description
Ctrl + \ Toggle PDF between fit to page / fit to width
Ctrl + [ Rotate PDF counter-clockwise 90°
Ctrl + ] Rotate PDF clockwise 90°

Developer

Unlock advanced features and debugging tools in Microsoft Edge with developer shortcuts, empowering you to optimize websites and enhance productivity during development.

Shortcut Description
F12 or Ctrl + Shift + I Open Developer Tools
Ctrl + Shift + R or Shift + F5 Reload the current page, ignoring cached content
Ctrl + U View source

Favorites

Shortcut Description
Ctrl + D Save your current webpage as a favorite
Ctrl + Shift + B Show or hide the favorites bar
Ctrl + Shift + D Save all open tabs as favorites in a new folder
Ctrl + Shift + O Open Favorites management
Alt + Shift + B Set focus on the first item in the favorites bar

After mastering these shortcuts, you'll navigate Microsoft Edge like a pro, boosting your productivity and efficiency. Remember to practice regularly to integrate them seamlessly into your workflow. Additionally, consider customizing shortcuts to suit your preferences and workflow better. Happy browsing!
