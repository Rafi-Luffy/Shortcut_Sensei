Top 75 Shortcuts for Telegram

Welcome to Shortcut Sensei!
Are you a frequent Telegram user looking to enhance your messaging efficiency? You're in the right place! Telegram is packed with shortcuts that can help you navigate chats, manage contacts, and customize your experience with ease.


Discover the Possibilities
In this guide, we've compiled the top 75 Telegram shortcuts, organized into categories for easy reference. Whether you’re replying to messages, sharing media, or adjusting settings, these shortcuts will streamline your Telegram interactions. Let’s dive in and discover how these shortcuts can transform your Telegram usage!


General Shortcuts

Quickly create new chats, groups, and channels with these essential Telegram shortcuts.

Key Combination Description
Ctrl + N New chat
Ctrl + Shift + N New secret chat
Ctrl + P New channel
Ctrl + Shift + P New group
Ctrl + M Minimize to tray
Ctrl + H View archived chats
Ctrl + Shift + N Open new Telegram window
Ctrl + Q Quit Telegram
Ctrl + W Close current chat

Navigation Shortcuts

Effortlessly move through your Telegram app with these shortcuts for quick access to chats, settings, and more.

Key Combination Description
Ctrl + Tab Switch to the next chat
Ctrl + Shift + Tab Switch to the previous chat
Alt + ↑ (Up Arrow) Move to the older chat
Alt + ↓ (Down Arrow) Move to the newer chat
Page Up Scroll up within the chat
Page Down Scroll down within the chat
Home Go to the beginning of the chat
End Go to the end of the chat

Chat Management Shortcuts

These are key combinations and quick commands that can help manage various chat-related tasks on Telegram, making communication more efficient and improving productivity.

Key Combination Description
Ctrl + Shift + K Mark chat as read/unread
Ctrl + E Edit selected message
Ctrl + F Search within the chat
Ctrl + K Clear the chat history
Ctrl + L Delete the chat
Ctrl + Enter Send a message

Message Actions Shortcuts

special commands and key combinations that help to interact with messages more efficiently on Telegram, making communication smoother and user experience more convenient.

Key Combination Description
Ctrl + R Reply to message
Ctrl + M Mark selected message as unread
Delete Delete selected message
Ctrl + Shift + M Mute notifications for the current chat
Ctrl + J Pin selected message
Ctrl + Shift + R Forward selected message
Ctrl + Shift + U Unarchive chat

Search and Filters Shortcuts

Quick commands and functions that enable users to efficiently search for specific content and apply filters to streamline their browsing and access to information on various platforms or applications.

Key Combination Description
Ctrl + Shift + F Advanced search
Ctrl + Shift + S Search chats
Alt + F4 Close current chat window
Ctrl + = (Equal) Increase chat text size
Ctrl + - (Minus) Decrease chat text size
Ctrl + 0 Reset chat text size

Contacts and Groups Shortcuts

Quick methods and functionalities that enable users to swiftly access and manage their contact lists and interact with various groups

Key Combination Description
Ctrl + G Open group from the list
Ctrl + Shift + D Add a contact
Ctrl + Shift + C Create a new contact
Ctrl + I Open contact info
Ctrl + Shift + I Share contact info
Alt + A Add member to a group
Alt + R Remove member from a group

File and Media Shortcuts

Commands and tools that help users quickly manage and interact with their files, such as documents, images, videos, and music, making it easier to navigate and organize digital content.

Key Combination Description
Ctrl + Shift + D Open your downloads folder
Ctrl + U Upload a file
Ctrl + Shift + U Upload media from the camera
Ctrl + D Download media/file
Ctrl + Shift + D Open media/file download folder
Ctrl + Shift + E Share location
Right Arrow (while viewing media/files) Go to the next media/file
Left Arrow (while viewing media/files) Go to the previous media/file

Call and Voice Messages Shortcuts

Features that allow users to swiftly initiate phone calls and interact through voice messages, simplifying communication and enhancing accessibility to these functions.

Key Combination Description
Ctrl + Shift + V Start voice call
Ctrl + Shift + V Start video call
Ctrl + . (Period) Record voice message
Alt + V Toggle video on/off in a call

Emojis, Stickers, and GIFs Shortcuts

These shortcuts enable you to effortlessly convey emotions and reactions, enhancing the overall engagement and fun quotient of your messaging sessions.

Key Combination Description
Ctrl + J Open emoji panel
Ctrl + G Open GIF panel
Ctrl + S Open stickers panel
Alt + E Switch between emoji, stickers, and GIFs

Profile and Settings Shortcuts

These shortcuts streamline the process of adjusting preferences and viewing/editing profile details, enhancing user control and navigation within the platform.

Key Combination Description
Ctrl + O Open/close settings
Ctrl + Y Change profile photo
Ctrl + Shift + A Address book
Ctrl + Shift + Z Clear search history
Ctrl + X Log out

Interface Shortcuts

These shortcuts enhance user experience by granting easy access to essential features, improving usability and accessibility.

Key Combination Description
Ctrl + W Close current window
F11 Enter/Exit Full-screen mode
Ctrl + Shift + M Minimize chat window
Ctrl + Shift + H Toggle hide sidebar
Ctrl + Shift + I Toggle developer tools
Ctrl + Shift + J Open JavaScript console
Ctrl + Shift + A Open the address book

Layout and Zoom Shortcuts

These shortcuts offer convenient ways to customize the visual presentation of content, ensuring a personalized and comfortable viewing experience.

Key Combination Description
Ctrl + + (Plus) Zoom in
Ctrl + - (Minus) Zoom out
Ctrl + 0 Reset zoom
Ctrl + B Toggle the left panel
