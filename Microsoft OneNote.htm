General Shortcuts

Essential shortcuts for common actions like creating, opening, saving, and editing notebooks.

Key Combination Description
Ctrl + N Create a new notebook
Ctrl + O Open an existing notebook
Ctrl + S Save changes to the current notebook
Ctrl + P Print the current page or notebook
Ctrl + Z Undo the last action
Ctrl + Y Redo the last undone action
Ctrl + A Select all items on the current page
Ctrl + X Cut the selected item
Ctrl + C Copy the selected item
Ctrl + V Paste the copied or cut item
Ctrl + F Open the Find dialog box to search for text
Ctrl + H Open the Replace dialog box to search and replace text
Ctrl + G Go to a specific page or section
Ctrl + Shift + F Open the Format Painter to copy formatting
Ctrl + B Bold the selected text
Ctrl + I Italicize the selected text
Ctrl + U Underline the selected text
Ctrl + Shift + N Create a new page
Ctrl + D Dock the OneNote window to the desktop
Ctrl + Alt + D Undock the OneNote window from the desktop
Ctrl + Alt + M Minimize the OneNote window to the taskbar
Ctrl + Alt + L Dock all OneNote windows
Ctrl + Alt + V Paste and match the formatting of the surrounding text
Ctrl + Alt + T Move focus to the Tag Summary pane

Formatting Shortcuts

Shortcuts for applying formatting styles, such as headings, font sizes, and tags, to your notes.

Key Combination Description
Ctrl + Shift + P Insert a page break
Ctrl + 1 Apply the Heading 1 style
Ctrl + 2 Apply the Heading 2 style
Ctrl + 3 Apply the Heading 3 style
Ctrl + Shift + > Increase the font size
Ctrl + Shift + < Decrease the font size
Ctrl + Alt + 1 Apply the To Do tag
Ctrl + Alt + 2 Apply the Important tag
Ctrl + Alt + 3 Apply the Question tag
Ctrl + Alt + 4 Apply the Star tag
Ctrl + Alt + 5 Apply the Critical tag
Ctrl + Alt + 6 Apply the Contact tag

Navigation Shortcuts

Handy shortcuts for navigating between pages, sections, and notebooks quickly.

Key Combination Description
Alt + Shift + N Open the navigation bar
Ctrl + Home Go to the first page in the current section
Ctrl + End Go to the last page in the current section
Ctrl + Alt + Home Go to the first page in the notebook
Ctrl + Alt + End Go to the last page in the notebook

Insert Shortcuts

Shortcuts for inserting new pages, sections, section groups, and tags into your notebooks.

Key Combination Description
Alt + N Insert a new page
Alt + H, N Insert a new section
Alt + H, P Insert a new section group
Alt + H, T Insert a new tag

View Shortcuts

Shortcuts for managing the appearance of your OneNote interface, such as toggling the navigation bar and docking page tabs.

Key Combination Description
Ctrl + M Toggle the navigation bar
Ctrl + Shift + Tab Switch between the notebook pane, section pane, and page pane
Ctrl + R Dock the page tab to the right side
Ctrl + L Dock the page tab to the left side
Ctrl + E Dock the page tab to the bottom
Ctrl + W Close the current notebook, section, or page

Tag Shortcuts

Shortcuts for applying specific tags to your notes, helping you organize and categorize your content efficiently.

Key Combination Description
Ctrl + Shift + 1 Apply the "Critical" tag
Ctrl + Shift + 2 Apply the "Contact" tag

Table Shortcuts

Shortcuts for working with tables, including moving between cells, merging and splitting cells, and more.

Key Combination Description
Tab Move to the next cell
Shift + Tab Move to the previous cell
Enter Move to the next row
Ctrl + Alt + M Merge selected cells
Ctrl + Alt + S Split selected cells

Drawing Shortcuts

Shortcuts for drawing and sketching in your notes, including converting handwriting to text and creating screen clippings.

Key Combination Description
Ctrl + P Convert handwriting to text
Ctrl + Space Show or hide the pens
Ctrl + 5 Create a screen clipping

Additional Shortcuts

Extra shortcuts for copying formatting, inserting dates and times, creating tables and pictures, and accessing the Help pane.

Key Combination Description
Ctrl + Shift + C Copy the formatting of the selected text
Ctrl + Shift + V Paste the formatting of the copied text
Alt + Shift + D Insert the current date
Alt + Shift + T Insert the current time
Ctrl + Shift + T Insert a table
Ctrl + Alt + P Insert a picture from a file
Ctrl + Shift + S Create a new subpage
Ctrl + Shift + E Create a new email message with the current page as an attachment
Ctrl + Alt + H Open the Help pane
