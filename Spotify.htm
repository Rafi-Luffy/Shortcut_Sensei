Navigation

Navigation shortcuts help you move around Spotify quickly, allowing you to jump between tracks, playlists, and search results with ease.

Key Combination Description
Ctrl + Up/Down Arrow Navigate to the previous/next track.
Ctrl + Left/Right Arrow Seek backward/forward within the current track.
Spacebar Play/Pause the current track.
Ctrl + Shift + Right Arrow Go to the next playlist.
Ctrl + Shift + Left Arrow Go to the previous playlist.
Ctrl + L Jump to the search bar.
Ctrl + F Start a new playlist.
Ctrl + Shift + F Create a new folder in your playlists.
Ctrl + N Start a new playlist.
Ctrl + Shift + N Create a new playlist folder.
Ctrl + Shift + C Copy a Spotify URI to the clipboard.
Ctrl + Shift + V Paste a Spotify URI from the clipboard.
Ctrl + R Refresh the current view.
Ctrl + Shift + R Refresh Spotify application.

Playback

Playback shortcuts give you control over the music playing in Spotify. You can play, pause, shuffle, repeat, and adjust playback settings effortlessly.

Key Combination Description
Ctrl + P Play current track.
Ctrl + S Shuffle playback.
Ctrl + T Repeat track.
Ctrl + Shift + T Turn off repeat.
Ctrl + Shift + L Toggle repeat between playlist, track, and off.
Ctrl + W Save a track to your library.
Ctrl + Shift + W Remove a track from your library.
Ctrl + J Go to the "Discover" section.
Ctrl + G Go to the "Genre & Moods" section.

Volume and Playback Control

These shortcuts enable you to manage the volume, skip tracks, and control playback speed, providing a customizable listening experience tailored to your preferences.

Key Combination Description
Ctrl + Shift + Up/Down Arrow Increase/decrease volume.
Ctrl + Shift + Left/Right Arrow Skip backward/forward in your queue.
Ctrl + Shift + Space Play the next song in your queue.
Ctrl + Shift + Left Click Select multiple tracks in your playlist.
Ctrl + Alt + Up/Down Arrow Adjust the pitch.
Ctrl + Alt + Left/Right Arrow Adjust the playback speed.

Search and Selection

Search and selection shortcuts streamline the process of finding and managing tracks in Spotify. You can select, copy, paste, and organize your music library efficiently.

Key Combination Description
Ctrl + A Select all tracks.
Ctrl + Z Undo your last action.
Ctrl + Y Redo your last action.
Ctrl + X Cut the selected track(s).
Ctrl + C Copy the selected track(s).
Ctrl + V Paste the selected track(s).
Ctrl + E Select the current track.
Ctrl + H Hide a track from your playlist.
Ctrl + Shift + H Unhide a track from your playlist.
Ctrl + M Mute playback.
Ctrl + Shift + M Unmute playback.
Ctrl + Alt + M Toggle mute.

Playlist Management

Playlist management shortcuts empower you to organize and customize your playlists effectively. You can open, share, queue, and edit playlists effortlessly.

Key Combination Description
Ctrl + O Open a playlist.
Ctrl + Shift + O Open a track.
Ctrl + U Share a playlist.
Ctrl + Shift + U Share a track.
Ctrl + K Select the current track's album.
Ctrl + Shift + K Select the current track's artist.
Ctrl + Q Queue the selected tracks.
Ctrl + Shift + Q Remove the selected tracks from the queue.
Ctrl + V Paste the copied tracks into the current playlist.
Ctrl + Shift + V Paste the copied tracks into a new playlist.
