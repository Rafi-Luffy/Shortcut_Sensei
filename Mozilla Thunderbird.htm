Email Navigation Shortcuts

Quickly navigate between different email folders such as Inbox, Sent, Drafts, etc., and move through email messages efficiently.

Key Combination Description
Ctrl + 1 Switch to Inbox folder
Ctrl + 2 Switch to Sent folder
Ctrl + 3 Switch to Drafts folder
Ctrl + 4 Switch to Junk folder
Ctrl + 5 Switch to Trash folder
Ctrl + 6 Switch to Archives folder
Ctrl + ↑ (Up Arrow) Move to the previous message
Ctrl + ↓ (Down Arrow) Move to the next message
Spacebar Scroll down in the message list
Shift + Spacebar Scroll up in the message list

Message Management Shortcuts

Manage email messages effectively by performing actions like creating new messages, replying, forwarding, deleting, and marking as junk or read.

Key Combination Description
Ctrl + N Create a new message
Ctrl + R Reply to the selected message
Ctrl + Shift + R Reply to all recipients of the selected message
Ctrl + F Forward the selected message
Ctrl + Shift + F Forward the selected message as an attachment
Ctrl + Shift + M Move the selected message to a different folder
Delete Delete the selected message
Ctrl + J Mark the selected message as junk
Ctrl + Shift + J Mark the selected message as not junk
Ctrl + Shift + U Mark the selected message as unread
Ctrl + Shift + K Mark the selected message as read
Ctrl + Shift + Delete Permanently delete the selected message

Composing Email Shortcuts

Streamline the email composition process with shortcuts for sending, saving drafts, inserting hyperlinks, and checking spelling.

Key Combination Description
Ctrl + Enter Send the composed message
Ctrl + S Save the composed message as a draft
Ctrl + Shift + S Save the composed message as a template
Ctrl + K Insert a hyperlink into the message
Ctrl + Shift + L Add a recipient from the address book
Ctrl + Shift + C Add a recipient from the contacts list
Ctrl + . (period) Insert a signature into the message
Ctrl + / (forward slash) Check spelling in the message

Search Shortcuts

Enhance productivity by searching for specific messages, text within messages, or messages from particular senders quickly.

Key Combination Description
