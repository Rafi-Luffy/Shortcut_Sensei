General Shortcuts

Discover essential shortcuts to swiftly navigate through File Explorer, including opening new windows, creating folders, and managing properties with just a few keystrokes.

Key Combination Description
Alt + D Select the address bar

Ctrl + F Select the search box
Backspace Go up one folder level in the hierarchy

Alt + Left Arrow Go back
Alt + Right Arrow Go forward
